## Key Design Decisions

### Travel Route Philosophy Shift

**Old Approach** (v0.0.05):
- Protagonist only visits 4 nations (Corinthia, Carnea, Salomia, Tobin)
- 5 nations remain unexplored (Aethelgard, Gilead, Niridia, Kanaan, Patmos)
- Simple ally/neutral paths reduce risk and complexity
- Story focused on political relationships, not cultural immersion

**New Approach** (v0.0.07):
- Protagonist visits all 9 nations in both acts (18 total stops across two journeys)
- First half: Mystery-driven detours and forced diversions create narrative tension
- Second half: Protagonist deliberately chooses the hardest path (enemy borders) as act of defiance
- Cultural immersion becomes core experience - every nation gets spotlight
- Justifies 3,000 levels: 120 lands = ~13 lands per nation × 9 nations
- Creates opportunity for deep linguistic and cultural storytelling per land

### Agent Formalization Rationale

**Why Create a Permanent Agent?**
1. **Reusability**: Language design work will be ongoing throughout content development (3,000 levels require consistent linguistic flavor)
2. **Consistency**: Single agent maintains design philosophy across all 9 nations
3. **Efficiency**: Pre-configured with context documents, research tools, and trigger conditions
4. **Collaboration**: Enables multi-agent workflows (story agents can invoke language agent when needed)
5. **Knowledge Preservation**: Codifies the linguistic framework as a reusable Claude Code resource

## Technical Integration Points

### StoryIntegration Service Updates Needed

Based on expanded travel routes, the StoryIntegration service (WORDRUN-AI-DEVELOPMENT-PLAN.md Section 6.2) must track:

```typescript
interface PlayerJourneyState {
  currentLand: number;        // 1-120
  currentNation: string;       // Which of 9 nations
  act: 1 | 2;                  // First half (delivery) or second half (crusade)
  nationsVisited: string[];    // Track 9-nation progression
  borderCrossings: {
    from: string;
    to: string;
    relation: 'ally' | 'neutral' | 'tense' | 'enemy';
    timestamp: Date;
  }[];
}
```

### LandTheme Schema Alignment

Each land now requires:
- **geographicNation**: Which of 9 nations (Aethelgard, Carnea, etc.)
- **journeyPhase**: 'first-half' | 'second-half'
- **borderType**: 'ally' | 'neutral' | 'tense' | 'enemy' (affects story tone)
- **languageRegion.mode**: 'A' (transformed) | 'B' (base) | 'C' (rebellion)
